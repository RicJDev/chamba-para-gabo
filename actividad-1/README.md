# Actividad para Tutores y Tutoritos - Algoritmos 1 y 2

Buenos días tengan todos, espero se encuentren excelentes, la presente es para dar de conocimiento que previo al inicio del nuevo semestre, quiero evaluar como los Tutores y Tutoritos van a estar trabajando, por ende, les voy asignar unas series de actividades para evaluar su desempeño y también, para que de una vez vayan trabajando juntos.

Cada tutor con su Tutoritos deberán hacer lo siguiente:

## 📚 Actividades por unidad

### Algoritmo 1
Deberán crear un enunciado y efectuar el pseudocódigo, por cada unidad de algoritmo 1, es decir, deberán crear y hacer el código para las siguientes unidades:

1. Introducción a los Algoritmos.
2. Estructuras Condicionales.
3. Ciclos repetitivos.
4. Sub algoritmos.
5. Vectores y matrices.

### Algoritmo 2

Deberán crear un enunciado y efectuar el pseudocódigo, por cada unidad de algoritmo 2, es decir, deberán crear y hacer el código para las siguientes unidades:

1. Arreglos.
2. Registros.
3. Arreglos de registros.
4. Arreglo de registros orientados a procesos sistemáticos (proyecto).


## 🧠 Modelos de exámenes

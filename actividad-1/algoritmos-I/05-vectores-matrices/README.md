# Vectores y Matrices. TEMA: , DIFICULTAD: , AUTOR: Ricardo

Un importante representante de payasos de circo nos ha encomendado hacerle una agenda para organizar la información de sus clientes. Valiéndonos de arreglos y sub algoritmos debemos crear un sistema que permita:

- Registrar payasos (con nombres, especialidad, número telefónico y circo contratante)
- Consultar la información detallada de cada payaso
- Modificar la información de un payaso en específico
- Borrar payasos
